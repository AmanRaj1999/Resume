Blog-Web-App
  • Developed using HTML, CSS, and JavaScript.
  • functionalities:
    • Fetch Data: The app fetches data from the JSONPlaceholder API (https://jsonplaceholder.typicode.com/posts) to retrieve a list of blog posts.
    • Display Blogs: The app dynamically creates HTML elements to display the fetched blog posts on the UI. Each blog post is shown with its title, content, and a delete button.
    • Add New Blog: Clicking on the "Add Blog" link reveals an input form where users can enter a title and content for a new blog post. Upon submitting the form, the app sends a POST request to the API to add the new blog post. The form is then cleared, and the updated list of blog posts is displayed on the UI, including the newly added blog post.
    • Delete Blog: Each blog post displayed on the UI has a delete button. Clicking on the delete button triggers a DELETE request to the API to remove the corresponding blog post. After successfully deleting the post, the UI is updated to reflect the changes by removing the deleted blog post from the list.
